# Quick Start Guide

## Option 1: Test Locally (Fastest)

### On Your Computer:
```bash
cd date-invite-h5
./serve.sh
```

Then open: **http://localhost:8080**

### On iPhone (same WiFi):
```bash
# The script will show your IP, like:
http://192.168.1.100:8080
```

Open that URL in Safari on iPhone.

---

## Option 2: Transfer to iPhone Directly

1. **AirDrop** the entire `date-invite-h5` folder to your iPhone
2. On iPhone, use the **Files** app
3. Find the `index.html` file
4. Tap it → Open in Safari
5. Done! ✨

---

## Option 3: Deploy to Web (Permanent)

### GitHub Pages (Free, 2 minutes):
```bash
# 1. Create new GitHub repository
# 2. Upload files from date-invite-h5 folder
# 3. Go to Settings → Pages
# 4. Enable GitHub Pages
# 5. Get URL like: https://yourname.github.io/date-invite/
```

### Netlify Drop (Free, 30 seconds):
1. Go to https://app.netlify.com/drop
2. Drag & drop the `date-invite-h5` folder
3. Get instant URL
4. Done! ✨

---

## Option 4: Add to iPhone Home Screen

1. Open the app in Safari
2. Tap **Share** button (bottom center, square with arrow)
3. Tap **"Add to Home Screen"**
4. Name it: **"Would You Go Out?"**
5. Tap **Add**
6. Now it's like a native app! 🎉

---

## Testing the App

- **Accept button**: Opens calendar picker
- **Reject button**: Moves away when you try to click it!
- Works completely offline
- No account needed
- No personal data collected

Enjoy! 💕
